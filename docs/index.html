🚀 Tất cả cách cài đặt
Chọn cách phù hợp với bạn
1

🪟 Windows Setup — Double-click (Đầy đủ nhất)

Tải ZIP → giải nén → double-click setup.bat. Script tự động cài VS Code + AGent Sweeps + Ollama + AI model cùng lúc — không cần làm gì thêm!

⬇️ Tải Windows Setup (.zip)

Trong .zip có: setup.bat + setup.ps1 — double-click setup.bat là xong

2

⚡ One-Click — VS Code URL (Nhanh nhất nếu đã có VS Code)

Click nút bên dưới, VS Code tự mở và cài trong 3 giây

⚡ Install in VS Code
3

🖥️ PowerShell 1 dòng (Windows — cài đầy đủ kể cả Ollama)

Mở PowerShell và chạy — tự cài VS Code + Extension + Ollama + model AI:

iwr -useb https://raw.githubusercontent.com/HoangGiaSeo/AGentSweeps/main/install.ps1 | iex
4

🐧 Bash 1 dòng (Linux / macOS / WSL — cài đầy đủ kể cả Ollama)

Mở Terminal và chạy:

curl -fsSL https://raw.githubusercontent.com/HoangGiaSeo/AGentSweeps/main/install.sh | bash
5

⌨️ VS Code CLI

Nếu VS Code đã cài và có trong PATH:

code --install-extension HoangGiaSeo.agent-sweeps
6

🔍 VS Code Marketplace Search

Mở VS Code → Extensions (Ctrl+Shift+X) → Tìm:

AGent Sweeps
7

☁️ GitHub Codespaces / Dev Container

Extension tự động cài khi mở repo này trong Codespaces hoặc Dev Container

# .devcontainer/devcontainer.json đã được cấu hình sẵn gh codespace create --repo HoangGiaSeo/AGentSweeps
8

📦 Từ file .vsix (Offline)

Tải file .vsix từ GitHub Releases rồi cài thủ công:

code --install-extension agent-sweeps-0.2.0.vsix

⬇️ Tải .vsix mới nhất tại đây
